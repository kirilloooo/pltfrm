FAQ

How long are the files stored? 

As of 12/22/2023, files are kept for 7 days, after which they are irrevocably deleted from all servers.

I found a spam file or compromising data on me or a friend, what should I do?

On the file page, there is a red "complain" button that you can use to notify a moderator to process the file.

What is the maximum file size?

Files are currently limited to 20 megabytes.
